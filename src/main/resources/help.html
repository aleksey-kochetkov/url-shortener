Installation

Place the jar file 
url-shortener-1.0.jar
as you wish.

Launching

java -jar url-shortener-1.0.jar

Usage

Assuming host name "localhost", use the following URL:
http://localhost:8080/account
to register new account via application/json request
http://localhost:8080/register
to register new url for the given account via application/json request, and providing Authorization header with Basic authentication token
http://localhost:8080/statistic
to get count of redirects for URLs of the given account via application/json request, and providing Authorization header with Basic authentication token
http://localhost:8080/{shortUrl}
to use registered shortening (you will be redirected as was set)
http://localhost:8080/help
displays this page in browser
